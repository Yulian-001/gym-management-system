Desarrollo de mi primer Sistema 

En este espacio se documentará el desarrollo de un sistema multiplataforma para la gestión de un gym.

Las herramienstas usadas son:

**BACKEND**
- Node.js
- Express.js
- PostgreSQL
- JWT
- bcryptjs
- Axios

El desarrollo multiplataforma se realizará con un mismo backend, tanto en el manejo de rutas, middleware, lógica de negocio, etc...

La aplicación para escritorio se realizará con: 

- Electron.js (Framework)
- React
- Axios
- CDD/SCSS

  La APK móvile se realizará con:
- React Native
- Expo (Aun no es definitivo)
- Axios
- React Navigation

Las herramientas seleccionadas son:
- JAVASCRIPT
- GITHUB
- REST API
- JWT Tokens

Las solicitudes del cliente son que el sistema sea totalmente intuitivo, su aprendizaje de uso sea fácil para distintos tipos de personas.

Esta es la maquetación del sistema, cumpliendo con las peticiones del cliente perfectamente, manteniendo la estética visual, la maquetación se hizo con la herramienta Figma, esta me permite adaptar el sistema enmultiplataformas, para diferentes dispositvos siendo muy eficiente.
link: https://www.figma.com/make/zFAFJuCh5hWpJX1dIYFxHm/Receta-App-de-Descubrimiento?node-id=0-1&t=QDplzD7g27Xi9DhZ-1

La paleta de colores seleccionada son tonos de azul claro con blanco.
